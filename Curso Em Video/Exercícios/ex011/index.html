Reproduzindo audio

Vamos aprender como reprozir audios no seu site, a estrutura é bem parecida com a de imagem

Quando falamos de audio, temos que ter sempre em mente a compatibilidade dele com os diferentes tipos de navegadores de madeira geral os Formatos mais aceitos sao MPEG, OGG e WAV. O ideal é sempre colocar no seu código se o arquivo estiver hospedado no seu servidor os diferentes formatos de arquivo, para assim obter maior compatibilidade. lembrando que a ordem do código usado nesse exercício é importante...... o ideal é sempre do mais compatível para o menos compatível, evitando assim carregamentos desnecessários. WAV é o menos útil dessa lista, pq WAV é mt pesado.

Tags utilizadas para audio:

  1. <Audio>
    • Preload ( uma das mais importantes, ele diz que o seu navegador vai carregar somente o basico do arquivo de audio )
    • Autoplay
    • Control
  2. <Source>
    • src
    • type
